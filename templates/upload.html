How to export the two reports (step-by-step)
Before you start

With the customer's authorization, open their RingCentral account (via SCP) and go to analytics.ringcentral.com.

  1. Open Analytics → Performance Reports and click the Calls tab.
  2. Set the date pill to your range (e.g. Last Month); leave all filter dropdowns at their defaults so nothing is excluded.
  3. Download → Excel and upload it as report 1 · PR Calls.
  4. Click the Queues tab, keep the same date range, Download → Excel, and upload as report 2 · PR Queues. Both are required — only the Queues file carries abandoned-in-queue counts, wait times, and service level.
PR Calls columns
Required: Session Id · Call Direction · Result · Call Length · Queue
Recommended: Handle Time · Call Start Time · From Number

The Calls tab includes these by default — if there's a Select Columns option, check all eight. The required ones gate the upload; the rest unlock the hourly chart, repeat-caller detection, and answered-under-60s. PR Queues has fixed columns — nothing to select.

Tip: the export is one row per call leg — the tool de-duplicates into one row per call, so no cleanup needed.
